# kataReact
React with express (TDD environment)

### GLOBAL DEPEDENCIES
 - NodeJs > 8.11.3
   
   ``` 
   $ curl -sL https://deb.nodesource.com/setup_8.x | sudo -E bash -
   $ sudo apt-get install -y nodejs
   ```
 
### DEPEDENCIES
 - Install [yarn](https://yarnpkg.com/en/) and execute:
  
  ```
  $ sudo npm install -g nodemon
  ```

### Run app:
yarn start

### Run eslint:
yarn eslint

### Run test:
yarn test
yarn testw
